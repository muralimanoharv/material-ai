## 挨拶エージェント（Greeting Agent）仕様書

挨拶エージェントは、Google ADK（Agent Development Kit）を使用したPythonベースの実装です。基本的な挨拶の処理、OAuth経由のユーザー識別情報の取得、遅延応答の管理、およびCSVアーティファクトの生成を行うように設計されています。

---

### 主な機能

このエージェントには、特定のユーザーの意図を処理するための専用ツールがいくつか備わっています。

* **標準的な挨拶:** 一般的な挨拶や問い合わせに対応します。
* **本人確認（ID取得）:** `oauth_user_details_context` を介してユーザーの詳細情報にアクセスします。
* **アーティファクト生成:** ID、氏名、役割（ロール）データを含むサンプルCSVファイルを作成し、保存します。
* **遅延応答:** 特定のキーワードで呼び出された際、応答前に10秒間の待機をシミュレートします。
* **エラーシミュレーション:** エラー処理と例外をテストするための専用ツールが含まれています。

---

### ツール定義

| 関数名 | トリガー / 説明 |
| :--- | :--- |
| `say_hello` | 標準的な挨拶ツール。 |
| `who_am_i` | 認証済みユーザーの詳細を取得。 |
| `create_csv` | `my-csv.csv` を生成し、ツールコンテキストに保存。 |
| `say_hi_10_after_seconds` | 応答前に実行を10秒間停止。 |
| `throw_error` | デバッグ用に例外を発生。 |

---

### 設定およびインストラクション

このエージェントは `gemini-3-flash-preview` モデルを使用しています。システム指示では以下の事項が優先されます。

1.  **直接的なツールマッピング:** 特定のフレーズが特定のツール呼び出しをトリガーします。
2.  **出力フォーマット:** エージェントは、Markdown形式を使用せず、シンプルなテキストで応答するように指示されています。
